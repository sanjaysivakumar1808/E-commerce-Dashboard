E-Commerce Sales Dashboard
📊 Project Overview

This project presents an interactive Power BI dashboard that provides insights into e-commerce sales data. It is designed to help businesses and analysts understand sales trends, product performance, regional sales distribution, and profitability at a glance.

🗂 Dataset

The dataset contains over 10,000 transactions.

Key fields include:

Order ID

Product Name

Category

Region

Order Date

Quantity

Profit

Sales



🔍 Analysis Approach

Data Cleaning & Preprocessing:

Converted date columns to proper DateTime format.

Handled missing values and ensured numeric columns are clean.

Exploratory Data Analysis (EDA):

Analyzed sales by category, region, and time.

Calculated profit margins and top-selling products.

Dashboard Creation in Power BI:

Added slicers for Category, Region, and Time.

Created visuals: bar charts, line charts, tables, and cards.

Enabled drill-through and interactive filtering for deeper insights.

💡 Key Insights from Dashboard

Top-selling Products: Identifies products generating the highest sales.

Regional Performance: Highlights regions contributing most to revenue.

Profit Analysis: Shows which products and categories are most profitable.

Sales Trends: Month-over-month sales trends help in forecasting and planning.
